## 🚀 التثبيت والتشغيل

### المتطلبات
- **Node.js** 18.0 أو أحدث
- **FFmpeg** (يُثبّت تلقائياً عبر npm)
- **الحساب على** WhatsApp (لمسح QR)

### 1. التثبيت المحلي

```bash
git clone https://github.com/Antigravity/Knightbot-MD.git
cd Knightbot-MD
npm install
```

### 2. إعداد البيئة

```bash
cp .env.example .env
# عدّل ملف .env وأدخل مفاتيح الـ API الخاصة بك
```

### 3. تشغيل البوت

```bash
# تشغيل عادي
npm start

# تشغيل بتطوير (إعادة تشغيل تلقائية عند التعديل)
npm run dev
```

### 4. تشغيل بـ Docker

```bash
# بناء وتشغيل
docker compose up -d

# عرض السجلات
docker compose logs -f

# إعادة بناء
docker compose build --no-cache
```

## ⚙️ متغيرات البيئة (.env)

| المتغير | الوصف | افتراضي |
|---------|-------|---------|
| `BOT_NAME` | اسم البوت | KnightBot MD |
| `BOT_PREFIX` | بادئة الأوامر | `.` |
| `OWNER_NUMBER` | رقم المالك (بدون +) | مطلوب |
| `OWNER_NAME` | اسم المالك | مطلوب |
| `GEMINI_API_KEY` | مفتاح Google Gemini | فارغ |
| `OPENROUTER_API_KEY` | مفتاح OpenRouter | فارغ |
| `LOLHUMAN_API_KEY` | مفتاح Lolhuman API | فارغ |
| `BETABOTZ_API_KEY` | مفتاح BetaBotz | فارغ |
| `LOG_LEVEL` | مستوى السجل | info |
| `MAX_VIDEO_DURATION` | أقصى مدة فيديو (ثواني) | 600 |

## 🛠️ التقنيات المستخدمة

- **[@whiskeysockets/baileys](https://github.com/WhiskeySockets/Baileys)** — مكتبة اتصال واتساب
- **[better-sqlite3](https://github.com/WiseLibs/better-sqlite3)** — قاعدة بيانات سريعة ومتزامنة
- **[yt-dlp](https://github.com/yt-dlp/yt-dlp)** — تحميل الفيديوهات
- **[sharp](https://sharp.pixelplumbing.com/)** — معالجة الصور
- **[fluent-ffmpeg](https://github.com/fluent-ffmpeg/node-fluent-ffmpeg)** — تحويل الوسائط
- **[@google/generative-ai](https://ai.google.dev/)** — Google Gemini API
- **[axios](https://github.com/axios/axios)** — طلبات HTTP

## ⚠️ التنبيهات

- **لا ترفع ملف `.env` أو مجلد `session/` لـ GitHub!**
- تأكد من إضافة رقمك في `OWNER_NUMBER` بملف `.env`
- للحصول على مفتاح Gemini المجاني: [Google AI Studio](https://aistudio.google.com/)
- للحصول على مفتاح OpenRouter: [OpenRouter](https://openrouter.ai/)
