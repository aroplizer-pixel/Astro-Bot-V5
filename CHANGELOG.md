# 📝 سجل التغييرات - KnightBot MD

## [2.0.0] - 2026-06-22

### 🔴 إصلاحات أمنية حرجة
- إزالة مفاتيح API المكشوفة من `config.js` ونقلها لـ `.env`
- إنشاء `.gitignore` شامل لحماية الملفات الحساسة
- إنشاء `.env.example` كنموذج للمتغيرات المطلوبة

### 🛡️ الأمان والاستقرار
- إضافة Global Error Handlers (`uncaughtException`, `unhandledRejection`)
- إضافة نظام إعادة الاتصال التلقائي مع Backoff تصاعدي
- إصلاح Bug: `sticker.js` كان يستخدم `commands` بدون استيرادها
- إصلاح Bug: `handler.js` كان يُشير لخصائص غير موجودة
- إصلاح: اسم موديل Gemini الخاطئ (قائمة fallback صحيحة)
- إضافة إشارات إيقاف آمنة (`SIGINT`, `SIGTERM`)

### 💾 قاعدة البيانات
- ترقية كاملة من JSON إلى **SQLite** (`better-sqlite3`)
- سكربت ترحيل تلقائي من JSON القديم إلى SQLite
- واجهة متوافقة (لا تكسر الإضافات الموجودة)
- فصل البيانات عن منطق العمل عبر الأعمدة JSON
- Prepared Statements للأداء العالي
- نسخ احتياطي يومي تلقائي
- إحصائيات مدمجة (`topWallet`, `topLevel`)

### 🏗️ تحسين البنية
- نظام سجل موسّع (`lib/logger.js`) مع مستويات (debug/info/warn/error) + حفظ ملفي
- دوال مساعدة مشتركة (`lib/utils.js`) لتقليل التكرار
- تحسين `metadataCache` بـ TTL تلقائي + تنظيف دوري
- استخراج نص الرسالة بـ `getMessageText()` موحد
- تحميل مسبق لطول الفيديو قبل التحميل الكامل
- تنظيف ملفات `temp/` القديمة عند الإقلاع
- توحيد كل `console.log` بـ `logger`

### ✨ ميزات جديدة
#### تحميلات جديدة
- تحميل من **فيسبوك** (`.فيسبوك`)
- تحميل من **تويتر/X** (`.تويتر`)
- تحميل من **تيليجرام** (`.تيليجرام`)
- البحث في يوتيوب (`.بحث_يوتيوب`)

#### ذكاء اصطناعي
- **ترجمة فورية** (`.ترجم`) — لأي لغة
- **تلخيص نصوص** (`.لخص`) — بالذكاء الاصطناعي
- تحسين موديلات Gemini بـ قائمة fallback

#### وسائط
- ملصقات فيديو متحركة عبر ffmpeg
- تحويل ملصق لصورة (`.صورة`)
- تحويل لـ MP3 (`.mp3`)
- تحويل لرسالة صوتية PTT (`.صوتي`)
- بيانات EXIF للملصقات (اسم الحزمة)

#### إدارة المجموعات
- معلومات الجروب التفصيلية (`.معلومات_الجروب`)
- تغيير اسم/وصف الجروب (`.تغيير_الاسم`, `.تغيير_الوصف`)
- جلب/تجديد رابط الدعوة (`.رابط_الجروب`, `.تجديد_الرابط`)
- قفل/فتح المجموعة (`.قفل_الجروب`, `.فتح_الجروب`)
- صورة بروفايل عضو (`.بروفايل_جروب`)

#### اقتصاد وألعاب
- قمار (`.قمار`) — 45% فرصة فوز
- سرقة (`.سرقة`) — 40% نجاح + نظام دروع
- تحويل أموال (`.تحويل`)
- درع حماية (`.درع`)
- لقب مخصص (`.لقب`)
- أسئلة ثقافية (`.ثقافة`)
- حجر ورقة مقص (`.حجر`, `.ورقة`, `.مقص`)
- ترتيب الأغنى (`.الاغنى`)

#### أدوات
- حالة الطقس (`.طقس`)
- حاسبة رياضية (`.حاسبة`)
- التاريخ والوقت (`.وقت`) — ميلادي + هجري
- تقصير روابط (`.رابط_قصير`)
- QR Code (`.qr`)

#### أوامر المالك
- إحصائيات البوت التفصيلية (`.stats`)

- إعادة تشغيل (`.ريستارت`)

- نسخ احتياطي فوري (`.نسخ`)
- قائمة الجروبات (`.قائمة_الجروبات`)
